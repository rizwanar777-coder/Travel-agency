# Travel Agency Webpage
This is a basic HTML project I created to practice building a simple webpage.
The page is about a travel agency that offers trips to Dubai.
## About the Project
The webpage includes:
* A short introduction about Dubai
* Different travel packages (group and private)
* Some images of popular places with captions
I Also added a meta description to understand how SEO works.
## What I Used
* HTML (only basics, no CSS or JavaScript)
## What I Learned
* How to structure a webpage using HTML
* How to use tags like `<figure>` and `<figcaption>`
* Adding links and images properly
* Basics of SEO using meta tags
## Improvements (if I continue this)
* Add CSS to make it look better
* Make the layout responsive
* Add more travel destinations
* Improve content and design
